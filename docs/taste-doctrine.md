## The Emotional Archaeology Doctrine

ÆRTA does not operate on similarity, but on **Resonance**.

### 1. Resonance vs. Similarity
Similarity is a metadata match (Genre: Sci-Fi $\rightarrow$ Genre: Sci-Fi). Resonance is a frequency match (Thematic Wound $\rightarrow$ Thematic Wound). We seek the "invisible thread" that connects a 17th-century poem to a modern film.

### 2. The Triptych of Taste
Taste is not a static profile, but a temporal movement:
- **The Core**: Permanent aesthetic axioms. The "North Star" of the user's taste.
- **The Tide**: The transient mood or current "inkling." The emotional frequency of the moment.
- **The Gap**: The space between the Core and the Tide. This is where the "Companion" operates, identifying what the user needs to balance their current state.

### 3. The Editor's Mandate
The system must avoid the "Feedback Loop" of mood-matching. It acts as an Editor, deciding between:
- **The Echo**: Validating the current Tide.
- **The Counterpoint**: Using the Core to stretch the user's attention, offering a work that resonates with the theme but shifts the emotional or formal perspective.
